# Data_Manipulation

##Introduction
This project was done as a part of my coursework for Data Manipulation. The propose of this project is to investigate and visualize the uber pickups data. The dataset contains over 4.5 million Uber pickups in New York City from April to September 2014. 

##Data
The files in the Dataset are separated by month and each has the following columns:
Date/Time: the data and time of the Uber pickup
Lat:The latitude of the Uber pickup
Lon:The longitude of the Uber pickup
Base:a code affiliated with the Uber pickup
